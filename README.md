WC 2026 Schedule — Live Score Site
A mobile-friendly World Cup 2026 schedule that shows all 104 matches in Pacific Time,
with live scores pulled automatically from openfootball's open data.
Features

All 104 matches with flags, channels (FOX/FS1), and PDT times
USA games highlighted in green
Scores fill in automatically after each match ends
Knockout bracket slots update as teams advance
Refreshes every 5 minutes
Works great on iPhone / Android

Deploy to GitHub Pages (5 minutes)
Step 1 — Create a new GitHub repository

Go to https://github.com/new
Name it worldcup2026 (or anything you like)
Set it to Public
Click Create repository

Step 2 — Upload the file

On the repo page, click Add file → Upload files
Drag index.html into the upload area
Click Commit changes

Step 3 — Enable GitHub Pages

Go to your repo Settings tab
Scroll to Pages in the left sidebar
Under Source, select Deploy from a branch
Set Branch to main, folder to / (root)
Click Save

Step 4 — Open your site
After ~60 seconds, your site will be live at:
https://YOUR-GITHUB-USERNAME.github.io/worldcup2026/
Bookmark that URL on your phone's home screen!
Add to iPhone home screen

Open the URL in Safari
Tap the Share button (square with arrow)
Tap Add to Home Screen
Tap Add

It will behave like an app — full screen, no browser chrome.
Data source
Scores come from openfootball/worldcup.json,
a free, open-source, no-API-key-required dataset maintained by the community.
Results typically appear within a few minutes to a few hours after a match ends.
The site auto-refreshes every 5 minutes when open.
